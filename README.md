# Simpli Tailwind

A SimpliWeb WordPress starter theme using Tailwind CSS v3. Built to be renamed and customised per project.

## Renaming the Theme

Before activating, rename the theme to suit your project:

1. Rename the theme folder (e.g. `simpli-tailwind` → `digital-cellar`)
2. Open `style.css` and update the theme header:

```css
/*
Theme Name: Your Theme Name
Description: Your description
Version: 1.0.0
Theme Author: SimpliWeb
Theme URI: https://simpliweb.com.au
*/
```

3. Update the constants in `functions.php` to match your project (see [Configuration](#configuration) below)

---

## Requirements

- Node.js 18+
- npm 9+
- Local by Flywheel (or any local WordPress environment)
- Advanced Custom Fields (ACF) — used by helpers

---

## Getting Started

After cloning, `node_modules/` and `assets/css/app.css` are not included. You'll need to install dependencies and run a build before the theme will render correctly.

### 1. Install dependencies

```bash
npm install -D tailwindcss postcss postcss-import autoprefixer @tailwindcss/forms @tailwindcss/container-queries
```

### 2. Build the CSS

```bash
npm run build
```

### 3. Activate the theme

In your WordPress admin go to **Appearance → Themes** and activate your renamed theme.

---

## Development

Watch for changes and rebuild automatically:

```bash
npm run watch
```

Source styles live in `src/input.css`. The compiled output is written to `assets/css/app.css` — do not edit this file directly.

## Configuration

Theme options are defined as constants in `functions.php`. Update these to suit your project before activating.

### Menus

```php
define('MENUS', [
    'primary' => 'Primary Navigation',
    'footer'  => 'Footer Navigation',
]);
```

### Image Sizes

```php
define('IMAGE_SIZES', [
    'card' => ['width' => 800, 'height' => 600, 'crop' => true],
]);
```

### Widget Areas

```php
define('WIDGETS', [
    'sidebar-1' => 'Primary Sidebar',
]);
```

### Logo Size

```php
define('LOGO_SIZE', [
    'width'  => 200,
    'height' => 60,
]);
```

## File Structure

```
simpli-tailwind/
├── inc/
│   ├── helpers.php        # Utility functions (auto-included)
│   └── setup.php          # Theme setup class (auto-included)
├── src/
│   ├── input.css          # Tailwind entry point
│   └── wordpress-core.css # WordPress block/admin styles
├── assets/
│   └── css/
│       └── app.css        # Compiled output (gitignored)
├── functions.php          # Constants, includes, enqueue
├── header.php
├── footer.php
├── index.php
├── style.css              # WordPress theme header only
├── tailwind.config.js
├── postcss.config.js
└── package.json
```

Any `.php` file placed in the `inc/` folder is automatically included — no manual require needed.

## Included Helpers

`inc/helpers.php` provides a set of utility functions available globally in templates:

| Function | Description |
|---|---|
| `simpli_schema_type()` | Outputs schema.org itemscope/itemtype for the body tag |
| `simpli_numbered_pagination()` | Renders numbered pagination on archive pages |
| `simpli_breadcrumbs()` | Returns an HTML breadcrumb trail |
| `get_time_ago($post_id)` | Returns a human-readable time ago string |
| `calculate_reading_time($post_id, $fields)` | Estimates reading time for a post |
| `acfimg($image, $size)` | Returns the URL from an ACF image array |
| `ifne($var, $index, $default)` | Safe array key access with fallback |
| `isarr($arr)` | Checks if a variable is a non-empty array |
| `phoneurl($number, $country_code)` | Converts a phone number to a `tel:` link |
